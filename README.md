# c-programs-smc

Small collection of C programs written for Santa Monica College (CS3).

## Programs

- **temperature_converter.c**
  - Converts Celsius to Fahrenheit
  - Converts Fahrenheit to Celsius
  - Prints conversion tables using loops and functions

- **calories_calculator.c**
  - Calculates total calories burned during a day
  - Supports running, walking, and biking
  - Uses user input and loops
